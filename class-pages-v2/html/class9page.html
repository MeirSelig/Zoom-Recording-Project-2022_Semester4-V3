Descpription

Class Description/Overview: In this class we go over making Forms in html. We learn about the Label tag and the Input tag. we also review responsive design from the previous class. We also Cover Semantic HTML:
Examples of non-semantic elements are: <div> and <span>, The Problem with these is that E-readers and Search Engines dont know what content is inside them.
Examples of semantic elements are: <nav>, <form>, and <article> these can be used in place of these non-semantic elements and will help with search engine optimization(not really sure what that means yet) also E-readers to understand where the content is within them.
The professor also said that when getting interviewed for a job you might get asked if you know Semantic html. which is very buzzwordy for such a simple thing just like Responsive Design. ____________.
